Où trouver Nethack ?

Malheureusement, Nethack n'est sans doute pas disponible chez votre revendeur habituel. Heureusement, il y a moyen de le télécharger sur le Net ! Vous craignez de vous retrouver sur un quelconque site de WaR3Z à l'honnêteté douteuse ? Pas de panique ! Nethack est entièrement gratuit, et c'est sur le site même des développeurs que vous pouvez l'obtenir. Vous y trouverez également le code source, si l'envie vous prend de créer votre propre version...

  • Le site officiel : www.nethack.org.
  • Télécharger directement Nethack 3.4.3 disponible pour Linux, MS-DOS, Windows 3.1, Windows 95, Windows 98, Windows NT, Windows 2000, Windows Me, Windows XP, Macintosh, Amiga, OS/2 et Atari (ouf !)

Mais, comme le code source est disponible et modifiable à volonté, certaines personnes ont créé d'autres interfaces de jeu pour Nethack : essayez-les si vous n'aimez pas celle disponible par défaut ! (N.B. : la liste ci-dessous n'est bien sûr pas exhaustive...)

  • La version Qt de Nethack pour Windows (version 3.3.1 seulement ; Nethack Qt - y compris 3.4.1 - est aussi disponible sous Linux sur le site officiel)
  • Falcon's eye, une version de Nethack en vue isométrique (sous Windows). Très joli, mais encore incomplet, et moins jouable qu'en 2D je trouve.
  • Nethack pour Psion (!)

Enfin, il existe une version de Nethack qui en reprend les concepts de base avec de nombreux ajouts (monstres, objets, classes, races, etc.) : Slash'em (SuperLotsaAddedStuffHack - Extended Magic). Plus riche, mais aussi moins équilibré que Nethack - une sorte de « terrain d'expérimentation » pour de futurs ajouts à Nethack.
