De l'aide, de l'aide !

Les pages d'aide ci-dessous sont conçues pour être utilisées directement : vous n'y trouverez pas ainsi par exemple la liste de tous les monstres et leur points de vie ou la description de l'algorithme utilisé pour déterminer vos chances de toucher un ennemi ; si cela vous intéresse, vous pouvez toujours jeter un coup d'oeil aux sources ou faire une recherche sur Google. Quand vous aurez lu ces spoilers, vous n'aurez plus aucune excuse (autre que l'inattention ou le manque de prévoyance) pour ne pas ascender !
